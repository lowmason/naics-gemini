Evaluate Training API¶

Comprehensive evaluation script for training stages. Extracts metrics from logs and TensorBoard, generates visualizations, and creates reports.

create_comparative_analysis(all_stage_results, output_dir) ¶

Create comparative visualizations across all stages.

evaluate_stage(stage, log_file, tb_dir, output_dir) ¶

Evaluate a single training stage.

extract_tensorboard_metrics(tb_dir, stage) ¶

Extract metrics from TensorBoard event files.

Returns:

Type Description
Dict[int, Dict[str, float]]

Dictionary mapping epoch -> {metric_name: value}

generate_report(all_stage_results, output_dir, configs) ¶

Generate comprehensive evaluation report.

load_json_metrics(metrics_file) ¶

Load evaluation metrics from JSON file.

main() ¶

Main entry point.

merge_tensorboard_metrics(log_metrics, tb_metrics) ¶

Merge TensorBoard metrics into log metrics.
